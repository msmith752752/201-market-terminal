# 📊 201 Market Terminal

A real-time financial market dashboard designed to deliver live data, price action, and data-driven insights across stocks, indices, and macroeconomic events.

This project powers live YouTube streams and is built with a focus on speed, clarity, and a modern “terminal-style” interface.

---

## 🚀 Features

- 📈 Live stock market tracking (price, % change, movers)
- 🌎 Macro and global market context
- ⚡ Real-time data visualization
- 🖥️ Clean, terminal-style dashboard UI
- 🔄 Designed for live streaming (OBS integration)

---

## 🧱 Tech Stack

- HTML
- CSS
- JavaScript
- (Planned) Python / FastAPI backend
- (Planned) API integrations for real-time financial data

---
